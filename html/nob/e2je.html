Noen ord begynner på э (i stedet for e)

Pronomenet этот 'den, det' og noen lånord begynner på э

Pronomenet этот 'den, den' i alle sine bøyningsformer begynner med bokstav э

Bøyningsmønstre til этот:

Kasus Hannkjønn Intetkjønn Hunnkjønn Pluralis
Nominativ этот это эта эти
Genitiv этого этого этой этих
Dativ этому этому этой этим
Akkusativ этот/этого это эту эти/этих
Instrumentalis этим этим этой этими
Lokativ этом этом этой этих

Ordet поэтому 'derfor' skrives med bokstav э

Noen lånord skrives med bokstav э på russisk

Eksempler på lånord som begynner på э: экзамен 'eksamen', экономика 'økonomi', этаж 'etasje'

Eksempler på lånord som skrives med э: поэзия 'poesi', неэффективность 'resultatløshet', мэр 'ordfører'
