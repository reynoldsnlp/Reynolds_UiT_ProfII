Spelling rule does not allow unstressed o after certain consonants

In unstressed syllables after the consonants ж, ц, ч, ш, щ we write e, not o

This rule is important for the following endings:

Nouns in the Instrumental Singular, where -ой -> -ей and -om -> -ем

Neuter in Nominative and Accusative Singular, where -oe -> -ee

Adjectives: Masculine and Neuter in Locative and Dative Singular, where -oм -> -eм and -oму -> -eму

Adjectives: Feminine in Genitive, Dative, Locative Singular, where -ой -> -ей

Adverbs where -о -> -е

Nominative Singular -> Instrumental Singular саботаж -> саботажем 'sabotage', абзац -> абзацем 'paragraph', царевич -> царевичем 'czar's son', товарищ -> товарищем 'comrade'

When the vowel is stressed, we write o: Nominative Singular -> Instrumental Singular нож -> ножом 'knife', отец -> отцом 'father', врач -> врачом 'doctor', борщ -> борщом 'borshch'

Masculine Nominative Singular -> Masculine Locative Singular хороший -> хорошем 'good'

When the vowel is stressed, we write o: Masculine Nominative Singular -> Masculine Locative Singular большой -> большом 'big'

In roots a stressed vowel after these consonants is often written with ё, as in: жёны 'wives', чёрный 'black', шёлк 'silk'

There are some frequent exceptions: шоколоад, шоссе, шофёр
